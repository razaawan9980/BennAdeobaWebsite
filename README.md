# Benn Adeoba Realtor Website

Ready static website for Benn Adeoba Realtor / Secret Agent Realty Inc.

## Publish
1. Open Netlify.
2. Drag the full `benn-adeoba-realtor` folder into Netlify deploy.
3. Or upload `benn-adeoba-realtor.zip`.

## Replace agent photo
The current file is a placeholder because the actual uploaded agent photo was not available inside this chat runtime.

To add Benn's real picture:
1. Save Benn's photo as: `assets/benn-adeoba-agent-photo.jpg`
2. Replace the existing file with the same name.
3. The website already crops it professionally using:
   - `object-fit: cover;`
   - `object-position: center top;`
   - rounded card and shadow styling.

## Update property listings
Open `script.js` and edit the `listings` array.
You can change:
- title
- price
- beds
- baths
- type
- size
- image
- description
- full description
- features
